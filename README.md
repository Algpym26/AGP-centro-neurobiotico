# AGP Centro Neurobiótico

Sitio web de AGP Centro Neurobiótico creado con React y Vite.

## Inicio rápido

1. Instala Node.js 20 o una versión más reciente.
2. En esta carpeta, ejecuta `npm install`.
3. Para iniciar el sitio en tu computadora, ejecuta `npm run dev`.
4. Para crear la versión lista para publicar, ejecuta `npm run build`.

Los recursos visuales están en `public/` y las secciones del sitio viven en `src/App.jsx`.

## Agregar un video nuevo

En `src/App.jsx`, agrega un elemento al arreglo `resources`. Para videos propios de YouTube, usa el tipo `youtube` y el identificador del video. Para un recurso externo, usa `link` y pega la URL.
